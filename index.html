Zee's Cheat Sheet

HTML

Elements
Name Tag Use Description
Opening <!DOCTYPE html> Put at the beginning of an html document to tell the browser what to expect
html <html>content here</html> Put around all of the html on the page. Opening tag should be just after DOCTYPE declaration, and closing tag should be the last thing in the file

CSS

Selectors
Selector Code Use Description Outranks in Specificity
Type p {
[Declarations here]
}

The above selector will select all p elements

 Can replace p with the content of any html tag to select that all such tags Will override inherited properties only
Descendant td p {
[Declarations here]
}

The above selector will select only those p elements which are inside td elements (table cells)

 Can replace td and p with any two elements to select for them instead. The first is always the parent, and the second is always the child Overrides inherited properties and Type selectors, so the td p selector overrides the p selector

Git & Bash

Bash
Command Example Usage Notes
Print Working Directory pwd Outputs the path to the current working directory from the base directory of the filesystem
Change Directory cd [path] Changes directory to the directory given in [path] which should be a relative path from the current working directory
Git
Command Example Usage Notes
Initiate git init Initiates a git repository in the current working directory
Add add [files paths]
add resources/css/index.css resources/photos/html.png

Adds both index.css and html.png to the staging area

 [files paths] should be a number of files or directories (or one) referenced by their relative paths from the current working directory and separated by spaces
